# Release Notes

## [v0.1.0]

### Added
- **Claude Code CLI Integration**: Added the ability to launch the Claude Code CLI directly from the workspace home screen. This opens a new terminal session running `claude`, providing seamless access to Claude's coding capabilities within the CodexOps environment.

### Changed
- **Terminal Integration**: Enhanced the terminal backend and frontend services to support passing initial commands to new terminal sessions. This facilitates features like the "Launch Claude Code" button.
- **Versioning**: Reset project versioning to v0.1.0 to mark this significant milestone.
